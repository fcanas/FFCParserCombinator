  • <<&(_:_:)

    Combines lhs and rhs into a new Parser that appends the results of the rhs to the results of lhs.

    This is very much like <&> with a 2-arity for lhs

    Declaration

    Swift

    public func <<& <S, A, B, C>(lhs: Parser<S, (A, B)>, rhs: Parser<S, C>) -> Parser<S, (A, B, C)>

    Parameters

    lhs

    A Parser of 2-element tuple that will match first, and whose results will be the first 2 elements of the returned Parser‘s results
    rhs

    A Parser of a single element that will match second, and whose results will be the last element of the returned Parser‘s results

    Return Value

    A parser matching lhs followed by rhs with results combined into a tuple in order.
